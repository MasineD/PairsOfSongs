# PairsOfSongs
Finding pairs of songs with total durations divisible by 60

## Introduction

    This project forms a solution to a challenge found on the Educative website. Click the following link to access the challenge [https://www.educative.io/daily-coding-challenge](https://www.educative.io/daily-coding-challenge)

## Problem description

You are given a list of songs, where the duration of each song is represented by an integer array time, where time[i] indicates the length of the i-th song in seconds.

Your task is to find and return the number of pairs of songs (i, j) such that:
    
    1. i < j (i.e., the pair should consist of two distinct songs)
    2.The sum of their durations is divisible by 60, i.e., (time[i] + time[j]) % 60 == 0.

Constraints:
    
    1. 1 ≤ time.length ≤ 1000
    2. 1≤ time[i] ≤ 500
